# plotEZ Examples

This directory contains various examples demonstrating PlotEZ functionality.

## Basic Examples

- `README_example1.py` - Simple line plot example
- `README_example2.py` - Dual y-axis plotting example
- `README_example2A.py` - Dual y-axis plotting example with customization
- `README_example3.py` - Multi-panel subplot example
- `README_example4.py` - Error bar plotting example

## Images

Generated plot images are saved in the `images/` directory.

## Running the Examples

```bash
# Run any example
python README_example1.py
python README_example2A.py

# Run the ErrorPlotConfig example
python README_example4.py
```

The examples demonstrate various `plotEZ` features and serve as both documentation and test cases for the library
functionality.
